This code is designed to analyze whole-genome long read sequencing data, without the requirement of extra dependencies. Our aim is to make this tool as user-friendly as possible so that the biologists with minimum coding experience can utilize it for the identification of the extrachromosomal circular DNAs (eccDNAs) from whole genomoe long-read sequencing data.

The input for the tool is a sorted bam file preferable mapped by minimap2 using the default parameters.
